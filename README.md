# 🤖 Projeto Android

Projeto desenvolvido durante meus estudos de **HTML5** e **CSS3**, acompanhando o curso de HTML e CSS do **Curso em Vídeo**, ministrado pelo professor Gustavo Guanabara.

O objetivo foi praticar a construção de páginas responsivas, organização semântica do HTML e estilização com CSS, utilizando como tema a história do mascote do Android.

---

## 📖 Sobre o projeto

Esta página apresenta curiosidades sobre a história do mascote do Android, utilizando recursos modernos de HTML e CSS para criar uma interface agradável e organizada.

Durante o desenvolvimento foram aplicados conceitos importantes de estruturação de páginas web, tipografia, cores, responsividade e organização do código.

---

## ✨ Funcionalidades

- ✅ Layout responsivo
- ✅ Navegação simples
- ✅ Estrutura semântica em HTML5
- ✅ Estilização utilizando CSS3
- ✅ Uso de imagens e vídeos
- ✅ Organização visual do conteúdo
